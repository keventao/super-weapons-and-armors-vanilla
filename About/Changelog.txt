# Changelog

All notable changes to this mod will be documented in this file.

## [1.3.0] - 2026-03-22

### Added
- Initial release of Vanilla (non-CE) version

### Changed
- Split from main CE mod into standalone Vanilla version
- Uses vanilla-native Bullet_FlyingTigersPlasma projectile instead of CE Bullet_ChargeRifle
- defNames include _Vanilla suffix to avoid conflict with CE version
- Armor and helmet: aligned with LWOP structure, full stat fields included
- Armor: Insulation_Heat=200, JumpRange=200, PackRadius=4.9, PsychicEntropyGain=0.0001
- Helmet: Insulation_Cold/Heat=200, VacuumResistance=100, ToxicEnvironmentResistance=99, full Royalty stats

## [1.2.1] - 2026-03-17 (via parent mod)

### Added
- HyperVitality Chip: Neural implant that reduces sleep need to 2 hours per day

## [1.2.0] - 2026-03-12 (via parent mod)

### Added
- Flying Tigers Elite Helmet
- Flying Tigers Elite Armor

## [1.0.0] - 2026-03-11 (via parent mod)

### Added
- Joywire implant
- Super Rifle (vanilla-adapted)
- Mod icon and preview images
