# Super Weapons and Armors

RimWorld mod that adds LWOP items with Combat Extended support.

## About

This mod is based on [LWOP Package](https://steamcommunity.com/sharedfiles/filedetails/?id=2012717312), adding Combat Extended compatibility to popular LWOP items.

## Requirements

- [RimWorld](https://store.steampowered.com/app/294100/RimWorld/) 1.6+
- [Combat Extended](https://steamcommunity.com/sharedfiles/filedetails/?id=2882957625)

## Current Features

### Weapons
- **Super Rifle**: High-capacity assault rifle (CE compatible)

### Implants
- **Joywire**: Neural implant that provides permanent mood boost

## Planned Features

- More LWOP items with CE support
- Balance adjustments
- New content

## Installation

1. Subscribe to [Combat Extended](https://steamcommunity.com/sharedfiles/filedetails/?id=2882957625) on Steam
2. Subscribe to this mod on Steam
3. Enable both mods in RimWorld (Combat Extended must load before this mod)

## Manual Installation

Copy this folder to your RimWorld mods directory:
```
<Steam>/steamapps/common/RimWorld/RimWorldMac.app/Mods/
```
