# Super Weapons and Armors - Vanilla

RimWorld mod that adds LWOP items **without** Combat Extended dependency.

## About

This mod is based on [LWOP Package](https://steamcommunity.com/sharedfiles/filedetails/?id=2012717312), adapted to work **without** Combat Extended.

**This is the non-CE version.** If you have Combat Extended installed, use the main "Super weapons and armors" mod instead.

## Requirements

- [RimWorld](https://store.steampowered.com/app/294100/RimWorld/) 1.6+
- No other dependencies required

## Current Features

### Weapons
- **Super Rifle**: High-capacity assault rifle (uses vanilla projectile)

### Implants
- **Joywire**: Neural implant that provides permanent mood boost
- **HyperVitality Chip**: Neural implant that reduces sleep need to ~2 hours per day

### Armor
- **Flying Tigers Elite Helmet**: Elite head protection
- **Flying Tigers Elite Armor**: Elite combat armor

## Installation

1. Subscribe to this mod on Steam (Vanilla version)
2. Enable it in RimWorld mod list (no load order requirements)

Or manually copy this folder to your RimWorld mods directory:
```
<Steam>/steamapps/common/RimWorld/RimWorldMac.app/Mods/
```

## CE vs Vanilla

| Feature | CE Version | Vanilla Version |
|---------|-----------|----------------|
| Combat Extended required | Yes | No |
| Projectile | CE Bullet_ChargeRifle | Vanilla native |
| defNames | No suffix | _Vanilla suffix |
